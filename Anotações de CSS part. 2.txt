Usando o id com CSS:Faz separação das configurações gerais ou tag.
 -Não use dois id no mesmo documento
 *Tudo que em HTML é id = em CSS é #
 *Tudo que em HTML é class = em CSS é .
Na pag HTML <h1 id="Nome da id">
Na pag CSS use #"Nome da id" criada em HTML

Para usar a mesma configuração em varios elementos use a tag "class"
 -Pode ser usada mais de uma class no mesmo elemento
 -Pode-se usar com id

Pseudo-classes:
 Usa-sa div para dividir uma configuração
 -':hover' - Para mostrar quando passar o mouse
 -':visited' - Mudar quando o link for clicado
 -':active' - Mudar no momento do click
